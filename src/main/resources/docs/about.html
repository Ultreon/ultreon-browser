This app has a different method for text editing, it can contain multiple editors in one window. This allows to manage the editors better without having windows spread everywhere on your screen. The way the app does it is by creating virtual windows inside the main window. Also, the app and editor can be customized, like themes. IntelliJ IDEA themes are supported.
